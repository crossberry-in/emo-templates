# CLAUDE.md

This file provides guidance to Claude Code when working with this emo project.

## Project overview

This is an emo app — built with the emo 0.1 SDK. The UI is written in `.em`
files (emo's custom language with JSX-like syntax) and styled with `.css`
files. The app runs on Android via Jetpack Compose.

## Key commands

- `emo start` — start the dev server with live reload
- `emo go` — launch on a connected Android device/emulator
- `emo build` — build a standalone APK
- `emo install <component>` — install a component from the registry

## File conventions

- `.em` files use JSX-like syntax: `<Column><Text>Hello</Text></Column>`
- `.css` files style elements via `className` attributes
- `app/` directory uses file-based routing (like Expo Router)
- `components/` contains reusable components
- `android/` contains the native Android project (Kotlin + Gradle)

## Do NOT

- Edit files in `android/app/build/` — they're generated
- Edit `__emo_gen__/` — generated by `emo build`
- Commit `.emo/` or `build/` directories
